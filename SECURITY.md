# Security Policy

## Supported Versions

The lastest version and versions released in the last 30 days.

| Version  | Supported          |
| -------  | ------------------ |
|   5.65.4 | :white_check_mark: |
| < 5.65.3 | :x:                |

## Reporting a Vulnerability

To report vulnerabilities please use the [Github security advisory system](https://github.com/crowbartools/Firebot/security). 
If you need assitance please see the [Discord server](https://discord.gg/crowbartools-372817064034959370)
